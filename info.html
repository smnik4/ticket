Основной функционал:

  • Ведение заявок;
  • Учет направлений (групп) заявок;
  • Распределение пользователей организации по направлдениям (каждый видит только свои направления);
  • Учет зданий (корпусов);
  • Отдельные типы заявок для ремонта оборудования, возможность печатать формы приемки оборудования.

Будет полезно для:

  • Отделы ИТ органицаций;
  • Организации по ременту и обслуживанию компьютерной и ОРГ техники;

Требования:

  • Не хранить собственные персональные данные или других лиц!
  • Не записывать информацию о аутентификации!

Варианты предоставления:

  • Общий пул - несколько организаций на одном хостинге;
  • Отдельный хостинг - свой на организацию.

Условия предоставления:

  • Бесплатно - при условии достаточности лимитов загрузки бесплатного хостинга и домена 3 уровня;
  • Бесплатно - Отдельный хостинг + свой домен любого уровня, при условии достаточности лимитов загрузки бесплатного хостинга и домена 3 уровня;

Плата за пользоватение системой не взимается.
